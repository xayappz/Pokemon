# Pokemon Cards

A new Pokemon project.
"Pokemon Cards" is a Flutter app designed to showcase a collection of Pok√©mon cards. It features smooth pagination, allowing users to browse through cards easily. Images are cached for fast loading, enhancing performance. The app boasts a beautiful, simple UI for an intuitive user experience.
